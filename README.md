# Portfolio

Go to terminal and install parsers:
```
npm i scraper-instagram
npm i linkedin-profile-scraper
```

And other tools which are important to install:
```
npm i fs
```

Parse social medias info using this commands:
```
node parseInsta.js
node parseLinked.js
```
